# FY-2026-27-UTN-Assessment
Repository for the San Joaquin County Unmet Transit Needs Assessment, including analysis and code used to identify transit-dependent residents whose essential travel needs are not adequately served by existing public transportation. Incorporates GIS mapping, ACS Census data, ridership/equity criteria, and policy evaluation methodologies.
